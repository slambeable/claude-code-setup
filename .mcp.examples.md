# MCP-серверы: примеры и рекомендации

В репозитории лежит `.mcp.json` с двумя серверами, которые я реально использую — Figma и Chrome DevTools. Ниже — описание, когда какой нужен, и как управлять ими в runtime.

## Правило подключения

Держи активным только то, что нужно прямо сейчас.

Каждый активный MCP грузит свои схемы в контекст при старте сессии. 3–5 серверов одновременно = 20–40% контекстного окна съедены ещё до первого вопроса. Это платные токены и деградация качества на длинных задачах.

## Как включать и отключать

`.mcp.json` задаёт список серверов, доступных проекту. Но включать/отключать их можно в runtime через UI Claude Code — без правки файла и перезапуска:

- `/mcp` — интерактивный список всех серверов с текущим статусом (enabled/disabled). Переключаешь нужный.
- `@mention` сервера в чате — альтернативный способ включить конкретный.
- `/context` — показывает, сколько контекста съедает каждый активный сервер. Удобно, когда чат замедлился, чтобы понять, кого пора отключить.

То есть типичный workflow: `.mcp.json` содержит всё, что может понадобиться проекту, но по факту активны только те серверы, которые нужны под текущую задачу. Перед длинной агентской сессией — прогоняй `/mcp` и выключай лишнее.

Требует Claude Code v2.0.10 или новее.

## Когда какой использовать

### Figma

Верстаешь компонент по макету — Claude читает файлы Figma, видит структуру фрейма, размеры, цвета, тексты. Надёжнее, чем «смотри на скриншот и угадывай».

### Chrome DevTools

Дебаг фронтенд-проблем: проверить, что в консоли, какие запросы в Network, что в DOM после клика. Claude сам открывает DevTools, читает вывод, предлагает правки.

## Текущая конфигурация

```json
{
  "mcpServers": {
    "figma": {
      "type": "http",
      "url": "https://mcp.figma.com/mcp"
    },
    "chrome-devtools": {
      "command": "npx",
      "args": [
        "chrome-devtools-mcp@latest",
        "--autoConnect"
      ]
    }
  }
}
```

## Где искать другие MCP-серверы

- Официальный реестр: https://github.com/modelcontextprotocol/servers
- Через команду `/mcp` внутри Claude Code — там же можно добавлять
